# NuGet Alternatives to SunamoLaTeX

This document lists popular NuGet packages that provide similar functionality to SunamoLaTeX.

## Overview

LaTeX document processing

## Alternative Packages

### LatexNET
- **NuGet**: LatexNET
- **Purpose**: LaTeX document generation
- **Key Features**: Create LaTeX documents programmatically

### MiKTeX.NET
- **NuGet**: MiKTeX.NET
- **Purpose**: LaTeX compilation
- **Key Features**: Compile LaTeX to PDF

### CliWrap
- **NuGet**: CliWrap
- **Purpose**: Execute LaTeX tools
- **Key Features**: Call pdflatex, xelatex commands

## Comparison Notes

Most LaTeX integration involves calling command-line tools via CliWrap.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **LatexNET**: LaTeX document generation
- **MiKTeX.NET**: LaTeX compilation
- **CliWrap**: Execute LaTeX tools
